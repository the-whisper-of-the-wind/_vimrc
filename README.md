# _vimrc

适用于windows下的vim/gvim,纯粹的文本编辑器(可以自行扩展成IDE）

+ 参考Ez大佬的配置
+ _vimrc带有详细的中文配置说明参考
+ 对插功能的深度配置

## 使用

在vim官网下载windows的vim，把仓库文件放在vim下
下载plug-vim放在autoload目录下,进入_virmc运行:PlugInstall
相同文件夹请把仓库文件夹和默认文件夹合并，相同文件替换为仓库文件

## 注意事项

仓库里的插件和vim文件大多为个人修改后的文件，请不要随便删除
